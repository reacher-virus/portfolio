Personal Portfolio Website

This repository contains my personal portfolio website built to showcase my skills, projects, and professional background. The website is fully responsive, interactive, and designed with a clean UI to present my work and experience effectively.

Tech Stack

HTML5 – Structure and semantic layout

CSS3 – Styling, animations, responsive design, and dark mode

JavaScript (Vanilla) – Interactivity and dynamic behavior

Libraries & Tools Used

Swiper.js – Services slider

MixItUp – Portfolio filtering and sorting

EmailJS – Contact form integration (no backend required)

Remix Icon – Icons

Features

Responsive layout for desktop and mobile devices

Interactive navigation menu with active section highlighting

Services section with slider

Skills section with progress indicators

Filterable portfolio/work section

Resume section with accordion-style layout

Working contact form with client-side validation using EmailJS

Light/Dark theme toggle and color switcher

How to Run Locally

Clone the repository

git clone https://github.com/your-username/your-repo-name.git


Open index.html in your browser

No build tools or server setup required.

Contact Form Setup

The contact form uses EmailJS.
To make it work with your own account:

Create an EmailJS account

Set up a service and email template

Replace the Service ID, Template ID, and Public Key in main.js
